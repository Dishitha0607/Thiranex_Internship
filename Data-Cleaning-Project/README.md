# 🎬 Netflix Data Cleaning & Visualization Project

## 📌 Overview
This project focuses on cleaning and analyzing the Netflix dataset to extract meaningful insights using Python. It demonstrates data preprocessing, handling missing values, and visualizing key trends in Netflix content.

---

## 🎯 Objectives
- Clean and preprocess raw Netflix dataset
- Handle missing values and duplicate records
- Perform exploratory data analysis (EDA)
- Visualize key insights using charts
- Understand content trends on Netflix

---

## 🛠️ Tools & Libraries Used
- Python 🐍
- Pandas 📊
- Matplotlib 📈
- Seaborn 🎨

---

## 📂 Dataset
The dataset contains information about Netflix movies and TV shows including:
- Title
- Type (Movie/TV Show)
- Country
- Date Added
- Rating
- Genre

---

## 🧹 Data Cleaning Steps
- Handled missing values using `fillna()`
- Removed duplicate records
- Converted `date_added` to datetime format
- Extracted `year_added`
- Split multi-value columns like `country` and `listed_in`

---

## 📊 Exploratory Data Analysis

### 1. Movies vs TV Shows
- Netflix has more Movies than TV Shows

### 2. Content Added Over Time
- Sharp increase in content after 2015
- Peak growth between 2018–2020

### 3. Top Countries
- United States dominates Netflix content
- India and UK are also major contributors

### 4. Top Genres
- International Movies and Dramas are most common
- Strong focus on global content

---

## 📈 Key Insights
- Netflix content is heavily dominated by Movies
- US produces the majority of Netflix content
- Drama and International Movies are the most popular genres
- Content growth peaked between 2018–2020
